# Juegos con PyGame

Aquí se almacenan los juegos desarrollados por el curso de Python Start 2 usando la librería PyGame.

* **Simple Paint:** Se genera un lienzo virtual en el que se puede dibujar con el ratón (clicks) o el teclado (flechas). El usuario puede elegir el color y grosor del trazo. Mejoras planeadas (en progreso):
  * Guardar el dibujo generado como un archivo.
  * Crear líneas punteadas y discontinuas.
  * Agregar formas simples (círculo, rectángulo, triángulo) previamente diseñadas.
